OCI(Oracle Cloud) 에서 console 연결하기

Cybertramp 2024-05-26
Categories： cloud Tags： setup

이전에 기록해둔 것인데… 다음에 사용할 때 참고하기 위해 옮겨 왔다.

나는 Free tier를 가끔 사용하는데 이때 인스턴스를 생성하고 콘솔에 연결하려면 생각보다 문서를 참고해도 무슨 말인지 파악이 어렵다… 인증서 기반으로 하다보니 더 까다롭게 접근해야하는 것같다.

처음에는 로그인이 불가능하다. 따라서 다음과 같이 적용하여 로그인 하고 암호를 변경한다.

Oracle linux8에서 확인한 내용이다.

  1. 재부팅 후 ESC 키 연타
  2. BIOS 화면에서 boot maintenance에서 boot from file 접근
    • redhat 디렉토리 내 grub64.efi 선택
  3. grub 화면에서 e 연타
  4. linux16으로 시작하는 부분에서 끝에 quiet LANG → quiet rw init=/bin/bash LANG 로 변경

  1. shell에 접근 가능해짐
    1. passwd opc
    2. touch /.autorelabel
    3. /usr/sbin/reboot –f
    4. 1번 단계에서 지정한 암호로 로그인
    5. opc로 접근시 암호로 접근 가능
