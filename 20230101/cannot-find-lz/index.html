cannot find -lz: No such file or directory

Cybertramp 2023-01-01
Categories： linux Tags： problem

빌드시 다음과 같은 에러가 발생하는 경우가 있다.

/usr/bin/ld:
collect2: error: ld returned 1 exit status
make: *** [makefile:243: Debug/NVMe_WorkLoad.exe] Error 1
make: *** [makefile:125: build_sub] Error 2

lz 라는게 없다고 이야기하고 있다.

-l 이야 lib 표시이고 z라는 라이브러리가 필요하다는 것이다. z는 zlib를 의미하는 것이고 나는 zlib 개발용을 받으면 된다.

설치는 다음과 같이 설치하면된다.

apt install zlib1g-dev -y
