install_wails

Cybertramp 2023-01-02
Categories： linux Tags： install

참고: https://wails.io/ko/docs/gettingstarted/installation

  1. install golang

    sudo apt install golang-go -y
export GO111MODULE=on
export PATH=$PATH:~/go/bin

  2. install gvm

    bash < <(curl -s -S -L https://raw.githubusercontent.com/moovweb/gvm/master/binscripts/gvm-installer)
source /home/{USERNAME}/.gvm/scripts/gvm

  3. install wails

    go install github.com/wailsapp/wails/v2/cmd/wails@latest

  4. check install is ok

    wails doctor

  5. make project

    wails init -n XXXX -t vue-ts
cd XXXX
cd frontend
yarn
cd ..

  6. Run Dev mode

    wails dev

  7. build with windows

    wails build -platform windows/amd64
